# MSMDLed
Mysensors led controller

Features:
* Control from MySensors
* Req sensors state
* Programe: Static color, Flash, Srrobe, Fade
* 24 buton IR Control 

Required libraries :
* [MySneosors](https://github.com/mysensors/MySensors/tree/master)
* [IRremote](https://github.com/z3t0/Arduino-IRremote)
